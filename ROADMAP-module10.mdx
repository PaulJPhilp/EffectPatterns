# Module 10: Data Types

Effect provides a rich set of immutable, type - safe data types that make functional programming in TypeScript practical and robust.  
Understanding these types is essential for modeling your domain, handling errors, managing state, and building composable programs.

This module introduces the most important data types in the Effect ecosystem, explains their purpose, and shows how to use them in real - world scenarios.
